## Overview

The Offers API allows you to create and manage promotional offers and discounts for both places and products. Offers can be configured with discount percentages, time periods, and associated with specific offer types.

## Authentication

All endpoints require JWT authentication via Bearer token. Different endpoints require different role permissions:
- **ADMIN**: Full access to all offer management
- **PLACE_ADMIN**: Can create and manage offers for their own places
- **USER**: Can view offers

## Offer Types

### Create Offer Type

<CodeGroup>
```bash cURL
curl -X POST https://api.dpm-delivery.com/offers/types \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Seasonal Discount"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.dpm-delivery.com/offers/types', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'Seasonal Discount'
  })
});
```
</CodeGroup>

**Endpoint:** `POST /offers/types`

**Required Role:** ADMIN

**Request Body:**
<ParamField body="name" type="string" required>
  The name of the offer type (e.g., "Flash Sale", "Seasonal Discount")
</ParamField>

**Response:**
```json
{
  "id": "uuid",
  "name": "Seasonal Discount",
  "createdAt": "2026-03-04T10:00:00Z",
  "updatedAt": "2026-03-04T10:00:00Z"
}
```

## Offers Management

### Create Offer

<CodeGroup>
```bash cURL
curl -X POST https://api.dpm-delivery.com/offers \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "reductionPercent": 20,
    "price": 1000,
    "start_date": "2026-03-04T00:00:00Z",
    "end_date": "2026-03-31T23:59:59Z",
    "offerType": "uuid-of-offer-type",
    "placeId": "uuid-of-place",
    "title": "Spring Sale",
    "description": "Get 20% off on all bookings this March"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.dpm-delivery.com/offers', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    reductionPercent: 20,
    price: 1000,
    start_date: '2026-03-04T00:00:00Z',
    end_date: '2026-03-31T23:59:59Z',
    offerType: 'uuid-of-offer-type',
    placeId: 'uuid-of-place',
    title: 'Spring Sale',
    description: 'Get 20% off on all bookings this March'
  })
});
```
</CodeGroup>

**Endpoint:** `POST /offers`

**Required Role:** ADMIN, PLACE_ADMIN

**Request Body:**
<ParamField body="reductionPercent" type="number" required>
  The discount percentage (e.g., 20 for 20% off)
</ParamField>

<ParamField body="price" type="number" required>
  The price after discount or minimum amount
</ParamField>

<ParamField body="start_date" type="string" required>
  ISO 8601 date string for when the offer starts
</ParamField>

<ParamField body="end_date" type="string" required>
  ISO 8601 date string for when the offer ends
</ParamField>

<ParamField body="offerType" type="string" required>
  UUID of the offer type
</ParamField>

<ParamField body="placeId" type="string">
  UUID of the place this offer applies to (optional, use for place-specific offers)
</ParamField>

<ParamField body="productId" type="string">
  UUID of the product this offer applies to (optional, use for product-specific offers)
</ParamField>

<ParamField body="title" type="string">
  Title of the offer
</ParamField>

<ParamField body="description" type="string">
  Detailed description of the offer
</ParamField>

**Response:**
```json
{
  "id": "uuid",
  "reductionPercent": 20,
  "price": 1000,
  "start_date": "2026-03-04T00:00:00Z",
  "end_date": "2026-03-31T23:59:59Z",
  "title": "Spring Sale",
  "description": "Get 20% off on all bookings this March",
  "type": {
    "id": "uuid",
    "name": "Seasonal Discount"
  },
  "place": {
    "id": "uuid",
    "name": "Beach Resort"
  },
  "createdAt": "2026-03-04T10:00:00Z",
  "updatedAt": "2026-03-04T10:00:00Z"
}
```

## Use Cases

### Creating a Place-Specific Offer

1. First, ensure you have an offer type created
2. Create an offer with `placeId` specified
3. Set the discount percentage and validity period
4. Users can view this offer when browsing the place

### Creating a Product-Specific Offer

1. Create an offer with `productId` specified instead of `placeId`
2. The offer will apply to that specific product
3. Can be used for product promotions or clearance sales

### Managing Seasonal Sales

1. Create an offer type like "Seasonal Sale"
2. Create multiple offers with different start and end dates
3. Update or delete offers as needed when campaigns end
