## Create Rider

<api-endpoint method="POST" path="/rider" />

Create a new rider account with vehicle and identification information. This endpoint creates both a user account with the COURIER role and the associated rider profile.

**Authentication:** Required

**Role Required:** Admin

**Content-Type:** `multipart/form-data` (required for file uploads)

### Body Parameters

#### User Information

<ParamField body="fullName" type="string" required>
  Rider's full name
</ParamField>

<ParamField body="phone" type="string" required>
  Rider's phone number
</ParamField>

<ParamField body="email" type="string" required>
  Rider's email address (required for riders, unlike regular users)
</ParamField>

<ParamField body="password" type="string" required>
  Initial password for the rider account (will be marked as default password)
</ParamField>

<ParamField body="profilePicture" type="file">
  Rider's profile photo (optional, image file)
</ParamField>

#### Vehicle Information

<ParamField body="bikeRegistrationNumber" type="string" required>
  Vehicle registration/license plate number
</ParamField>

<ParamField body="bikeType" type="string" required>
  Type of delivery vehicle (e.g., "Bike", "Aboboyaa", "Van")
</ParamField>

<ParamField body="bikeColor" type="string" required>
  Vehicle color
</ParamField>

<ParamField body="bikeBrand" type="string" required>
  Vehicle manufacturer/brand (e.g., "Honda", "Yamaha")
</ParamField>

<ParamField body="bikeModel" type="string" required>
  Vehicle model
</ParamField>

<ParamField body="bikeYear" type="number" required>
  Vehicle manufacturing year (e.g., 2020)
</ParamField>

<ParamField body="bikeImage" type="file" required>
  Photo of the vehicle (image file)
</ParamField>

#### Identification Documents

<ParamField body="identificationDocumentNumber" type="string" required>
  ID document number (driver's license, national ID, etc.)
</ParamField>

<ParamField body="identificationDocumentType" type="string" required>
  Type of ID document (e.g., "Driver's License", "National ID")
</ParamField>

<ParamField body="documentExpiryDate" type="string (ISO 8601)" required>
  Document expiration date (format: YYYY-MM-DD)
</ParamField>

<ParamField body="identificationDocumentImage" type="file" required>
  Photo of the identification document (image file)
</ParamField>

### Example Request

```bash
curl -X POST "https://api.dpm.com/rider" \
  -H "Authorization: Bearer <admin_token>" \
  -F "fullName=Kwame Mensah" \
  -F "phone=+233501234567" \
  -F "email=kwame.mensah@example.com" \
  -F "password=TempPassword123" \
  -F "bikeRegistrationNumber=GH-1234-20" \
  -F "bikeType=Bike" \
  -F "bikeColor=Red" \
  -F "bikeBrand=Honda" \
  -F "bikeModel=CB125F" \
  -F "bikeYear=2022" \
  -F "identificationDocumentType=Driver's License" \
  -F "identificationDocumentNumber=DL-456789" \
  -F "documentExpiryDate=2026-12-31" \
  -F "bikeImage=@/path/to/bike-photo.jpg" \
  -F "identificationDocumentImage=@/path/to/license.jpg" \
  -F "profilePicture=@/path/to/profile.jpg"
```

## Update Rider

<api-endpoint method="PUT" path="/rider/:id" />

Update an existing rider's information including vehicle and document details.

**Authentication:** Required

**Role Required:** Admin

**Content-Type:** `multipart/form-data` (supports file uploads)

### Path Parameters

<ParamField path="id" type="string (UUID)" required>
  The rider's user ID to update
</ParamField>

### Body Parameters

All fields are optional. Only include fields you want to update.

<ParamField body="bikeRegistrationNumber" type="string">
  Vehicle registration/license plate number
</ParamField>

<ParamField body="bikeType" type="string">
  Type of delivery vehicle
</ParamField>

<ParamField body="bikeColor" type="string">
  Vehicle color
</ParamField>

<ParamField body="bikeBrand" type="string">
  Vehicle manufacturer/brand
</ParamField>

<ParamField body="bikeModel" type="string">
  Vehicle model
</ParamField>

<ParamField body="bikeYear" type="number">
  Vehicle manufacturing year
</ParamField>

<ParamField body="bikeImage" type="file">
  Updated photo of the vehicle
</ParamField>

<ParamField body="identificationDocumentNumber" type="string">
  ID document number
</ParamField>

<ParamField body="identificationDocumentType" type="string">
  Type of ID document
</ParamField>

<ParamField body="documentExpiryDate" type="string (ISO 8601)">
  Document expiration date
</ParamField>

<ParamField body="identificationDocumentImage" type="file">
  Updated photo of identification document
</ParamField>

<ParamField body="profilePicture" type="file">
  Updated profile photo
</ParamField>

### Example Request

```bash
curl -X PUT "https://api.dpm.com/rider/rider-user-id-123" \
  -H "Authorization: Bearer <admin_token>" \
  -F "bikeRegistrationNumber=GH-5678-21" \
  -F "bikeColor=Blue" \
```

## Get Rider Statistics

<api-endpoint method="GET" path="/rider/:id/stats" />

Retrieve performance statistics for a specific rider. Riders can only view their own statistics.

**Authentication:** Required

**Role Required:** Courier (the rider must be requesting their own stats)

### Path Parameters

<ParamField path="id" type="string (UUID)" required>
  The rider's user ID
</ParamField>

### Example Request

```bash
curl -X GET "https://api.dpm.com/rider/rider-user-id-123/stats" \
  -H "Authorization: Bearer <rider_token>"
```

## Document Expiry Management

<Tip>
  The system tracks document expiry dates. It's recommended to:
  - Monitor riders with documents expiring within 30 days
  - Prevent assignment of new deliveries to riders with expired documents
  - Send reminders to riders to update their documents before expiry
</Tip>

### Example: Check for Expiring Documents

While there's no dedicated endpoint, you can filter the rider list and check `documentExpiryDate`:

```javascript
const today = new Date();
const thirtyDaysFromNow = new Date(today.getTime() + (30 * 24 * 60 * 60 * 1000));

const ridersWithExpiringDocs = riders.filter(rider => {
  const expiryDate = new Date(rider.rider.documentExpiryDate);
  return expiryDate <= thirtyDaysFromNow && expiryDate >= today;
});
```
