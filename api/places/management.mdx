## Create Place

Create a new place (vendor location) with a dedicated place admin.

```http
POST /places
```

**Authentication**: Required (Bearer token)  
**Authorization**: `ADMIN` role only

### Request Body

<ParamField body="name" type="string" required>
  Place name
</ParamField>

<ParamField body="email" type="string" required>
  Contact email
</ParamField>

<ParamField body="phone" type="string" required>
  Phone number (Ghana format: +233...)
</ParamField>

<ParamField body="address" type="string" required>
  Physical address
</ParamField>

<ParamField body="longitude" type="string" required>
  Geographic longitude coordinate
</ParamField>

<ParamField body="latitude" type="string" required>
  Geographic latitude coordinate
</ParamField>

<ParamField body="website" type="string">
  Business website URL
</ParamField>

<ParamField body="category" type="string" required>
  Category ID (UUID)
</ParamField>

<ParamField body="averagePrice" type="number" required>
  Average item price
</ParamField>

<ParamField body="deliveryFee" type="number">
  Delivery fee in local currency
</ParamField>

<ParamField body="minPrepTime" type="number">
  Minimum preparation time (minutes)
</ParamField>

<ParamField body="maxPrepTime" type="number">
  Maximum preparation time (minutes)
</ParamField>

<ParamField body="logo" type="file" required>
  Logo image file (multipart/form-data)
</ParamField>

<ParamField body="mainImage" type="file" required>
  Cover/hero image file (multipart/form-data)
</ParamField>

<ParamField body="placeAdminFullName" type="string" required>
  Full name of the place administrator
</ParamField>

<ParamField body="placeAdminPhone" type="string" required>
  Phone number of the place administrator
</ParamField>

<ParamField body="placeAdminPassword" type="string" required>
  Password for the place administrator account
</ParamField>

### Response

## Update Place

Update place details. Super admins can update any place, place admins can only update their own.

```http
PUT /places/:id
```

**Authentication**: Required (Bearer token)  
**Authorization**: `ADMIN` or `PLACE_ADMIN` roles

### Path Parameters

<ParamField path="id" type="string" required>
  Place ID (UUID)
</ParamField>

### Request Body

All fields are optional:

<ParamField body="name" type="string">
  Place name
</ParamField>

<ParamField body="email" type="string">
  Contact email
</ParamField>

<ParamField body="phone" type="string">
  Phone number (Ghana format)
</ParamField>

<ParamField body="address" type="string">
  Physical address
</ParamField>

<ParamField body="longitude" type="string">
  Geographic longitude
</ParamField>

<ParamField body="latitude" type="string">
  Geographic latitude
</ParamField>

<ParamField body="website" type="string">
  Business website URL
</ParamField>

<ParamField body="category" type="string">
  Category ID (UUID)
</ParamField>

<ParamField body="averagePrice" type="number">
  Average item price
</ParamField>

<ParamField body="deliveryFee" type="number">
  Delivery fee
</ParamField>

<ParamField body="minPrepTime" type="number">
  Minimum preparation time (minutes)
</ParamField>

<ParamField body="maxPrepTime" type="number">
  Maximum preparation time (minutes)
</ParamField>

<ParamField body="logo" type="file">
  New logo image (multipart/form-data)
</ParamField>

<ParamField body="mainImage" type="file">
  New cover image (multipart/form-data)
</ParamField>

### Response

Returns the updated place object.

## Update Opening Hours

Set or update the weekly schedule for a place.

```http
POST /places/:id/openingHours
```

**Authentication**: Required (Bearer token)  
**Authorization**: `ADMIN` or `PLACE_ADMIN` roles

### Path Parameters

<ParamField path="id" type="string" required>
  Place ID (UUID)
</ParamField>

### Request Body

Each day can have one of three states:

1. **Closed**: `{ "closed": true }`
2. **Open All Day**: `{ "openAllDay": true }`
3. **Time Ranges**: `{ "ranges": [{ "from": "09:00", "to": "17:00" }] }`

<ParamField body="monday" type="object" required>
  Monday opening hours
</ParamField>

<ParamField body="tuesday" type="object" required>
  Tuesday opening hours
</ParamField>

<ParamField body="wednesday" type="object" required>
  Wednesday opening hours
</ParamField>

<ParamField body="thursday" type="object" required>
  Thursday opening hours
</ParamField>

<ParamField body="friday" type="object" required>
  Friday opening hours
</ParamField>

<ParamField body="saturday" type="object" required>
  Saturday opening hours
</ParamField>

<ParamField body="sunday" type="object" required>
  Sunday opening hours
</ParamField>

<ParamField body="exceptions" type="array">
  Special hours for holidays/events
</ParamField>

### Example Request

```json
{
  "monday": {
    "ranges": [
      { "from": "09:00", "to": "17:00" },
      { "from": "18:00", "to": "22:00" }
    ]
  },
  "tuesday": {
    "ranges": [
      { "from": "09:00", "to": "22:00" }
    ]
  },
  "wednesday": { "openAllDay": true },
  "thursday": { "ranges": [{ "from": "09:00", "to": "22:00" }] },
  "friday": { "ranges": [{ "from": "09:00", "to": "23:00" }] },
  "saturday": { "openAllDay": true },
  "sunday": { "closed": true },
  "exceptions": [
    {
      "name": "Christmas Day",
      "date": "2024-12-25",
      "repeating": true
    }
  ]
}
```

<Note>
Time formats must use 24-hour military time (HH:MM). For example: "09:00", "14:30", "23:59".
</Note>
